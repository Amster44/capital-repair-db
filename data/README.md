## Формат файлов в каждой папке региона

Для каждого региона необходимо скачать 3 файла с сайта https://фонд-кр.рф/opendata

### Обязательные файлы:

1. **export-kr1_1-XX-YYYYMMDD.csv** - Многоквартирные дома в региональной программе капитального ремонта
   - Содержит: адреса домов, площади, количество жителей, способ формирования фонда (спецсчет/регоператор), финансовые данные

2. **export-kr1_2-XX-YYYYMMDD.csv** - Конструктивные элементы, инженерные системы и объекты общего имущества МКД
   - Содержит: данные о лифтах (тип, количество остановок, даты ввода/вывода), кровли, фасады, системы отопления и т.д.

3. **export-kr1_3-XX-YYYYMMDD.csv** - Услуги (работы) по капитальному ремонту МКД
   - Содержит: историю выполненных работ, подрядчиков, стоимость, даты выполнения

где:
- `XX` - код региона (например, 16 для Татарстана)
- `YYYYMMDD` - дата выгрузки (например, 20260201)

## Инструкция по загрузке данных

1. Перейдите на https://фонд-кр.рф/opendata
2. Выберите нужный регион из списка
3. Скачайте все 3 отчета (КР 1.1, КР 1.2, КР 1.3) в формате CSV
4. Сохраните файлы в соответствующую папку региона
5. Запустите скрипт импорта: `python scripts/import_csv.py --region XX`

## Примечания

- Файлы в формате CSV с разделителем `;` (точка с запятой)
- Кодировка: UTF-8 with BOM
- Десятичный разделитель: `,` (запятая)
- Убедитесь, что названия файлов соответствуют шаблону выше
